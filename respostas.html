  3. Uma aplicação web é um "Software baseado nas tecnologias e nos padrões do World Wide Web Consortium (W3C) que fornece
    recursos específicos através da Web, como conteúdos e serviços, através de uma interface de usuário, o navegador Web".
    De acordo com essa definição, responda: Web sites, sem componentes de software, como páginas HTML estáticas, são consideradas
    aplicações Web? Justifique.
    • Não. Porque os web sites sem componentes não possuem serviços que fornece uma interface para o usuário navegar, exemplo os sites estáticos para empresas onde pode ocupar um espaço na internet apenas para trazer informações simples sob a mesma. Diante isso podemos afirmar que os web sites sem componentes não fornece recursos específicos na web para suprir as necessidades do usuário, então não é uma aplicação web.
